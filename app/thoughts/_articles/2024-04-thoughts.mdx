# 2024年4月碎碎记：丢掉幻想，拥抱AI与现实

丢掉幻想，准备战斗。

四月的思考，是一场从内观心法到外部世界的全面探索。它关于信心，关于定义，也关于如何在一个充满变化的时代立体地看世界。

## 内观与心法

本质是信心。一个人越来越强大的迹象是：**情绪稳、能力稳、行事稳、心态稳、内核稳**。

### 学习一个新领域的三板斧

1.  **关键词拓展**：用 ChatGPT 输出 100 个领域的关键词，并搞懂它们。
2.  **经典阅读**：读该领域的专业教科书，从导论开始，逐步展开，10 本足够。
3.  **专家交流**：找到该领域的专家，并与他们交流。

### 思考的快慢系统

我们的思考模式可以分为两个系统，理解它们有助于做出更好的决策：

- **System 1 (快思考)**: 快速、直觉、自动、情绪化。它不费力气，比如 `2 + 2 = ?`
- **System 2 (慢思考)**: 缓慢、理性、有意识、逻辑化。它需要努力，比如 `17 x 24 = ?`

### 提升思考能力的10种心智模型

1.  **第一性原理思考**：从头开始重新思考问题。
2.  **二阶思考**：不仅考虑直接后果，更要考虑第二阶段的后果。
3.  **反转法**：从终点而不是起点看待问题。多问“我应该避免什么？”
4.  **机会成本**：选择一个选项时，要考虑到因此而放弃的其他选项的成本。
5.  **随机性**：请记住，并非所有事情都有因果关系。
6.  **杠杆**：“只要给我足够长的杠杆，我就可以移动世界。”
7.  **安全边际**：为你的错误假设规划一个安全边际。
8.  **奥卡姆剃刀法**：“如无必要，勿增实体。”总是选择最简单的解释。
9.  **边际效用递减**：到达某个点后，额外单位带来的价值会减少，而成本可能增加。
10. **专业领域**：利用专业化，选择一个你能成为专家的领域。

### 幸福五法则

1.  不要恨任何人。
2.  不要和别人比较。
3.  不要为过去或未来过度担忧。
4.  不要指望付出便能得到回报。
5.  不要抱怨你的困难。

## 创业与博弈

沧海横流，方显英雄。底层拼出来的英雄，往往对现实有更深刻的理解。

### 创业 vs. 做生意

- **做生意**：常被形容为“人傻、钱多、速来”。
- **创业**：通常意味着一个新技术、新的商业模式。

创业需要经常自省，避免自我强化和催眠。CEO 的角色很容易陷入周围少有批评的困境。此时，自知、自制和反省能力变得尤为重要，能从轻微的意见中发现问题并修正，这是一种慎独。

### 关于钱的思考框架

对任何事情，都可以按这个思路想一想：
1.  **是否涉及钱？**
2.  **谁花钱？谁赚钱？**
3.  **从花钱角度看**：有什么收益？是主动还是被迫？
4.  **从赚钱角度看**：这钱你想赚也能赚吗？还有谁能赚？怎么才能赚到？

### 权利运行的逻辑

权利运行的逻辑往往是：**盲目服从，被驱使，没有自己的主见。**

### 管理能力七步法

1.  **有工作，就有目标** (销售额、费用、利润)
2.  **有目标，就有计划** (课题、标准、时间、成果、责任人)
3.  **有计划，就有执行** (培训、讲解、分解)
4.  **有执行，就有检查** (建立检查小组)
5.  **有检查，就有结果** (通报检查战报)
6.  **有结果，就有责任** (区分好坏)
7.  **有责任，就有奖惩** (培养人才，树立企业文化)

## AI Agent：迈向智能的未来

GenAI 已经第三年了。我们正在从简单的模型调用，走向更复杂的 Agent 系统。

### Agent 进展综述

- **架构**：从单 Agent 到多 Agent 系统。
- **关键因素**：推理、规划、工具调用。
- **示例**：
  - **单 Agent**: ReAct, RAISE, Reflexion
  - **多 Agent**: AgentVerse, MetaGPT, DyLAN
- **未来研究**：Agent 评估、现实世界适用性。

### 如何利用 AI 打造个人“智囊团”

AI 作为个人助手的最佳实践：
1.  **Organizer** (组织者)
2.  **Reminder** (提醒者)
3.  **Connector** (连接器)
4.  **Expander** (扩展器)
5.  **Deep diver** (深度探索者)
6.  **Test-and-iterator** (测试与迭代者)

### 写作指南

最后，附上 George Orwell 的六条写作规则，它们永不过时：
1.  绝不使用印刷物里常见的修辞。
2.  能用短词，就别用长词。
3.  能删掉的词，一定删掉。
4.  能用主动语态，就别用被动。
5.  能用日常词，就别用外来词、科学术语或行话。
6.  为了不违反上述任何一条，可以打破任何规则。
